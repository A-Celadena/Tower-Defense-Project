# Tower Defense Game

## Project Description

This Tower Defense game is a Java project built using BlueJ. It features a classic wave-based defense experience where the player places towers to stop enemies from reaching the end of the path. The game uses object-oriented design with inheritance to model towers and enemies, making it easy to extend with new behaviors and balance changes.

Players can choose from three tower types:

- **Basic Tower**: A standard defense tower with balanced rate of fire and damage.
- **Rapid Tower**: A fast-firing tower designed to handle groups of weaker enemies.
- **Sniper Tower**: A long-range tower that deals high damage to single targets.

Enemies include three distinct types:

- **Grunt**: A basic enemy with average speed and health.
- **Speedster**: A fast enemy that can quickly close distance.
- **Tank**: A heavily armored enemy with high health and lower speed.

## Technologies Used

- Java
- BlueJ IDE
- Object-oriented programming with inheritance
- Custom classes for tower and enemy behavior

## How to Run

1. Open the `Tower-Defense-Project` folder in BlueJ.
2. Compile all Java classes.
3. Run the main game class from BlueJ.
4. Use the in-game controls to place towers and defend against enemy waves.

> Note: If any BlueJ-specific project files are included, make sure the project is opened directly in BlueJ to preserve the class interactions and launch process.
